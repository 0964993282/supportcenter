Hackathon listings page

This page lists all upcoming hackathons on the Devpost platform. The default sort order is by submission deadline, so hackathons with the submission period coming up soonest are displayed at the top of the list. You can filter the list by hackathon type (online or in-person) or use the search box for a keyword search, which indexes text from the hackathon’s title, location, domain name, tagline, and description (on hackathon homepage).

Each listing row contains the hackathon’s...

  • Thumbnail image
  • Title
  • Location (city, state, and country if in-person hackathon; “online” text if online hackathon)
  • Tagline
  • Total prize value
  • Submission period dates

From the top header bar of this page, you also can access the page to view all projects or learn more about posting a hackathon.
